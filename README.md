# Automated UI Exploration Demo


This repository provides a demonstration video of an automated UI exploration system,
showcasing its ability to traverse mobile application interfaces and capture ad-related views.

## Description


The demonstrated system automatically explores application interfaces without human intervention.
It continuously navigates UI states and covers ad-related views during the exploration process.


This repository is intended for demonstration and visualization purposes only.
